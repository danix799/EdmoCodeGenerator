# EdmoCodegenerator ![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)

EdmoCodegenerator is a database centric template based code generator for any text(ascii) programming language like C, PHP, C#, Visual Basic, Java, Perl, Python... Supported databases are SQL Server, MySQL (Coming soon Postgre, Oracle and Sqlite )

## Installation

### Requirements
* Windows 7 or higher
* Net Framework 4.5 or higher

## Usage

* Add database
* Add template directory
* Click the "Generate!" button
